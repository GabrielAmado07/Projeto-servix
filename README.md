# Servix 🚀

## 📌 Sobre o Projeto

O **Servix** é uma plataforma web desenvolvida para conectar trabalhadores autônomos e informais a clientes que buscam serviços de forma prática, segura e transparente.

O projeto foi criado com foco em acessibilidade, simplicidade e experiência do usuário, oferecendo páginas de cadastro, login, compras e pedidos em uma interface moderna e responsiva.

---

# 🎯 Objetivos

* Facilitar a conexão entre prestadores de serviço e clientes.
* Criar uma plataforma intuitiva para contratação de serviços.
* Oferecer uma experiência visual moderna e responsiva.
* Simular um ambiente de marketplace de serviços.

---
